# Katelyn Cook Fine Art

Production-ready Next.js 16 website for fine art painter Katelyn Cook.

## Stack
- Next.js 16 (App Router, Turbopack)
- Tailwind CSS v4
- Supabase (database + CRM)
- TypeScript

## Getting Started

```bash
npm install
cp .env.local.example .env.local  # add your Supabase keys
npm run dev
```

## Setup
Run `supabase-schema.sql` in your Supabase SQL Editor to create all tables.
